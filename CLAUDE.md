## Sobre el portafolio
Portafolio web en HTML/CSS/JS hosteado en GitHub Pages.
Muestra proyectos de Unity/C#, VFX en tiempo real y skills de diseño.

## Referencias de diseño
Inspirarse en estos tres portafolios para decisiones visuales:
- https://hugo.fyi → estructura limpia, proyectos bien organizados
- https://gerlogu.com → gamedev técnico, muestra sistemas y proyectos
  indie con detalle, similar al perfil de René
- https://janreeh.com → estilo visual impactante, composición limpia

## Estilo objetivo
- Colorido y creativo, con personalidad — no genérico
- Fondo oscuro con acentos de color vibrantes
- Cards para proyectos con imagen/GIF + descripción + tags
- Tipografía bold con carácter
- Navegación simple: Projects / VFX / Skills / Contact
